# phishing-detector
Данный проект посвящен разработке высокоточной системы классификации фишинговых веб-сайтов. Вместо классических деревьев решений здесь используется TabNet - инновационная архитектура от Google Cloud AI, которая объединяет преимущества глубокого обучения и интерпретируемость древовидных моделей.
