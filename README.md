# Detecting Phishing Websites with Attentive Interpretable Tabular Learning (TabNet)

## О проекте 

Этот репозиторий посвящен разработке интеллектуальной системы обнаружения фишинговых веб-сайтов на основе анализа 30 ключевых признаков (URL, SSL-сертификаты, структура страницы). Основная цель проекта - создание интерпретируемого и высокоточного классификатора угроз.

В основе решения лежит **TabNet** - инновационная архитектура от Google Cloud AI, которая объединяет преимущества глубокого обучения и интерпретируемость деревьев решений.

### Ключевые особенности реализации:
*   **Sequential Attention:** В отличие от классических нейросетей, TabNet использует механизм последовательного внимания для выбора наиболее значимых признаков на каждом шаге принятия решения.
*   **Unsupervised Pre-training:** Применен этап самообучения (`TabNetPretrainer`) на всем массиве данных для извлечения скрытых закономерностей в структуре URL без использования разметки.
*   **Bayesian Optimization:** Гиперпараметры модели оптимизированы с помощью **Optuna**, что позволило найти идеальный баланс между сложностью нейросети и ее обобщающей способностью.
*   **Data Refinement:** Проведен глубокий EDA, в ходе которого были удалены неинформативные признаки с нулевой корреляцией, что снизило зашумленность данных.

Проект демонстрирует эффективность применения **Deep Learning** для табличных данных в задачах кибербезопасности, обеспечивая не только высокую точность, но и прозрачность принимаемых решений.

### Ключевые инсайты:
*   **Преимущество Pre-training:** Самообучение на структуре данных без учителя позволило модели лучше обобщать признаки, что дало стабильный прирост точности.
*   **Интерпретируемость (Feature Importance):** Благодаря архитектуре TabNet удалось визуализировать наиболее значимые признаки. Топ-факторами при определении фишинга стали:
    1. `SSLfinal_State` (Наличие и доверие к SSL-сертификату).
    2. `URL_of_Anchor` (Анализ ссылок внутри страницы).
    3. `Prefix_Suffix` (Наличие дефисов в доменном имени).
*   **Стабильность:** Модель показала высокую устойчивость к переобучению, сохранив минимальный разрыв между валидационной и тестовой точностью.
